# 3D Patch Composition

Read when one continuous surface would collapse visible roles into a smooth
sheet, bowl, ribbon, two mounds, or isolated extremum.

First list visible roles: shelves, cavities, rims, side folds, high/low masses,
crag fields, broken perimeters, or overlapping layers. Build a small set of
projected patches in that order, then match camera, box aspect, subject scale,
and front/back depth before adding local texture.

Connect patches through folds, overlaps, shared material lighting, and partial
occlusion. Use `plot_surface` only as one role among others, not as the whole
object plus decorative sides.

Reject patch output that becomes a support/enclosure, one extremum replacing
many roles, a long slab, random triangles, or one gridded top skin. Preserve
accepted camera, occupancy, material class, and export floor.
